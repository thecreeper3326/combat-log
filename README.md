## No Combat Log

This mod is a small server-side utility that lets you set various practical (and fun) punishments.

### Punishments:
- Health : have the player join back with a certain amount of health
- Hunger : have the player join back with a certain amount of hunger
- Saturation : drain the player's saturation when they join
- Slowness Effect : give the player slowness when they join
- Hunger Effect : give the player hunger when they join, to keep them from sprinting
- Poison Effect : give the player poision when they join, to keep them at 1 HP
- Blindness Effect : inhibit the player's ability to see

### Extras:
- NoGravity : gives the player a crazy amount of levitation
- Nausea : gives the player nausea
- Fire : sets the player on fire
- Scale : sets the player's scale to a given multiplier

## Configuration

All functionallity is completely server-side so vanilla clients can join and play like normal. If by chance you want to configure the mod through a GUI, install it onto a client with operator permissions and run `/combat_log_config`
If you do not want to install the mod on your client, just go to the server's config directory and look for a folder named `combat_log`. Inside it will be the config file itself and an info document to help make sense of what each option does.
